SceneForge Export Tool

Paste GPT storyboard output → Export to PowerPoint or InDesign CSV

How to use this tool

  1. Run your script through the SceneForge GPT and ask for markdown table output.
  2. Copy the full table (including the header row) and paste it below.
  3. Click Parse Scenes to preview your storyboard.
  4. Export to .pptx for presentation or .csv for InDesign Data Merge.
Step 1

Paste SceneForge GPT output

Paste the markdown table from your SceneForge GPT session. The table must include a header row with pipe-delimited columns.

Step 2 — Preview

Storyboard scenes 0

Review parsed scenes before exporting. Each card shows scene metadata and the Firefly prompt.

Step 3 — Export

Export your storyboard

Choose your output format. PowerPoint creates one slide per scene. CSV is formatted for InDesign Data Merge.
